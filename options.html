General - General Twitch options

Type your Twitch username in the input field below in order to fetch a list of streamers you're currently following (while the list itself will be shown through the extension by clicking the icon in the right hand corner of your browser).

Tweaking - Tweak the values to your own likings (or don't)

Set the refresh rate of the counter in the right hand corner of the browser (in minutes). The changes won't apply untill you chrome://restart/. Anything less than 1 minute is unstable.
minute(s)
Set the max limit of users fetched using the Twitch API. Please be cautious and don't call unnecessarily to the API - we don't want complaints from Twitch, thanks.
user(s) fetched
Set the duration of the animation when loading the list of streamers (in seconds). Disable the animation completely by specifying 0 seconds.
second(s)
